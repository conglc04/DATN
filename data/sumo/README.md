# SUMO Bạch Mai MCI Scenario

OSM network + ambulance convergence traces for W15 (M10 mobility).

## Setup (one-time)

```bash
# 1. Install SUMO
sudo apt-get install -y sumo sumo-tools

# 2. Download OSM data (Bạch Mai 5km×5km, Overpass API)
python3 01_download_osm.py

# 3. Convert OSM → SUMO network
bash 02_build_network.sh

# 4. Generate ambulance routes
python3 03_generate_routes.py

# 5. Run SUMO simulation → FCD XML
bash 04_run_simulation.sh
```

## Files

| File | Description |
|------|-------------|
| `bachmaiHN.osm` | Raw OSM data (generated by step 1) |
| `bachmaiHN.net.xml` | SUMO network (generated by step 2) |
| `ambulance_routes_k1.xml` | K=1 trips (generated by step 3) |
| `ambulance_routes_k3.xml` | K=3 trips (generated by step 3) |
| `bachmaiHN_mci_k1.fcd.xml` | **K=1 FCD trace** (generated by step 4) |
| `bachmaiHN_mci_k3.fcd.xml` | **K=3 FCD trace** (generated by step 4) |
| `generate_traces.py` | Synthetic fallback (no SUMO needed) |

## Scenario

- **gNB**: Bạch Mai Hospital entrance (21.002966°N, 105.840780°E) — SSOT: `utils/config.BACH_MAI_LAT/LON`
- **Cell radius**: 300m
- **K=3 MCI convergence**: 3 ambulances at different bearings, all converge to gNB
- **Speed**: 60 km/h (hard-mission default)
- **Fidelity**: Tầng 1 — OSM road geometry + traffic signals, no traffic calibration
